🔑 Token GitHub Personnel

Sans token : 60 req/heure.
Avec token : 5 000 req/heure.

1 Aller sur github.com/settings/tokens
2 Cliquer Generate new token (classic)
3 Cocher uniquement public_repo
4 Coller le token ci-dessous

🔒 Le token est stocké uniquement dans votre navigateur (localStorage).
Il n'est jamais envoyé ailleurs que vers api.github.com.
